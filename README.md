# Projet DevOps 2022 - Exoplanets

## Description

Our 2022 devops project based on the exoplanets.

### Prerequisites

-   [Node JS / NPM](https://nodejs.org/en/)

### Installing

Install dependencies

```
npm i
```

## Getting Started

Run the app

```
npm start
```
